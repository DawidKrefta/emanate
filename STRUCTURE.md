# Project Structure Documentation

## eMANate Component Structure

### Main Components:
- **Header** (`/src/components/Header.tsx`) - Navigation with logo, menu, CTA button
- **Hero** (`/src/components/Hero.tsx`) - Main section with hero message and CTA
- **Features** (`/src/components/Features.tsx`) - Services section with feature cards
- **About** (`/src/components/About.tsx`) - Clinic space description
- **PriorityAccess** (`/src/components/PriorityAccess.tsx`) - Sign-up form

### Folders:
- `/src/components/` - all React components
- `/src/assets/` - images, icons, media
- `/src/styles/` - global styles, Tailwind config

### Tech Stack:
- React + TypeScript
- Tailwind CSS for styling
- Vite as bundler
- Bun for package management

### Color Palette:
- Dark Teal: `#1a2e2a`
- Medium Teal: `#2d4a45`
- Cream: `#f5f0e8`
- Beige: `#e8dfd0`
- Gold: `#c9a961`

### Fonts:
- Display: Outfit (headings, bold text)
- Body: Inter (paragraphs, UI text)
